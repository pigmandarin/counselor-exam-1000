# 상담심리사 1급 모의 문제은행 1000제

상담심리사 1급 필기시험 출제기준(5과목)에 맞추어 자체 제작한 모의문항 1,000문항 학습 도구입니다.
실제 기출문제를 포함하지 않으며, 전 문항이 AI(Claude) 보조로 새로 제작되었습니다.

- **바로 풀기**: https://pigmandarin.github.io/counselor-exam-1000/
- 구성: 5과목 × 200문항, 실전 모의고사 7회차(각 125문항), 전 문항 해설 포함
- 채점 기준: 실제 시험과 동일 (과목 40점 미만 과락 + 전체 평균 60점 이상 합격)

## 오류 제보
